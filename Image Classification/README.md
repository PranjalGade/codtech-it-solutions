# Image Classification using CNN

## 📌 Project Overview
This project implements an Image Classification system using a Convolutional Neural Network (CNN). The model is trained to classify images into different categories by automatically learning visual features from the dataset.

## 🎯 Task Objective
- Build an image classification model using Deep Learning
- Train a Convolutional Neural Network (CNN)
- Evaluate model performance
- Perform predictions on sample images

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Jupyter Notebook

## ⚙️ Workflow
1. Dataset loading and preprocessing
2. Image normalization and resizing
3. CNN model architecture design
4. Model training and validation
5. Model evaluation
6. Prediction on new images

## 📂 Project Files
- `Image Classification.ipynb` – Main implementation notebook
- Dataset folder
- Model output & predictions

## 📊 Result
The CNN model successfully learns image patterns and performs classification with good accuracy after training.

## 🚀 Learning Outcomes
- Understanding CNN architecture
- Image preprocessing techniques
- Model training & evaluation
- Deep learning workflow implementation
