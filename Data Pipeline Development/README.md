# Data Pipeline Development

## 📌 Task Overview
This project implements a **Data Pipeline Development** to automate data preprocessing, transformation, and loading using Python. The pipeline prepares raw data for analysis and machine learning applications.

## 🎯 Task Objective
- Extract raw data
- Clean and preprocess dataset
- Transform features
- Load processed data for further use

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## ⚙️ Pipeline Steps
1. Data Extraction using Pandas  
2. Data Cleaning and Handling Missing Values  
3. Feature Transformation and Scaling  
4. Saving Processed Data

## 📂 Project Files
- `Data Pipeline development.ipynb` – Main notebook
- etl_pipeline
- Processed output data

## ✅ Result
Developed an automated ETL pipeline that converts raw data into a clean and machine-learning-ready dataset.
